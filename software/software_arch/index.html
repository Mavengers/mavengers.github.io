软件架构

核心架构体系

驴车的整体架构是通过在 Linux 系统(目前是 Ubuntu 20.04 LTS版本)上构建，通过本地 Python 环境及LattePanda, OpenCV, Tensorflow, Keras, OpenVINO, Arduino，以及云端Azure Machine Learning, Azure DevOps等云服务整合而成。
